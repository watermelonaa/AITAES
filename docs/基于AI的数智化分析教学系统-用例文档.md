## []()UC02管理教师账号

| - **项目** | - **内容** |
| --- | --- |
| - ****用例编号**** | - UC02 |
| - ****用例名称**** | - 管理教师账号 |
| - ****参与者**** | - 系统管理员 |
| - ****前置条件**** | - 系统管理员已登录 |
| - ****后置条件**** | - 教师账号信息被创建、修改或状态变更 |
| - ****基本流程**** | 1.系统管理员进入“教师账号管理”页面2.查看教师账号列表（工号、姓名、账号、状态等）3.执行以下任一操作： a.手动添加教师：填写工号、姓名、初始密码 b.批量导入教师：下载模板后上传Excel文件 c.禁用/启用账号：点击操作按钮并确认 d.重置密码：设置新密码或生成随机密码 e.编辑教师信息：修改姓名（工号不可修改） f.搜索教师：按姓名或工号模糊搜索4.系统保存变更并刷新列表5.用例结束 |
| ****业务规则**** | 系统管理员不接触任何教学业务数据；工号为唯一标识，不可修改 |
